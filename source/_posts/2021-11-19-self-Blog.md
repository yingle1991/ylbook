百度告诉我 attr

<!-- more--> 

```
<view content="19" class="_span">19</view>

._span {
		font-size: 150upx;
		font-weight: 900;
		color: black;
		font-family: 'Courier New', Courier, monospace;
	}

	._span::before {
		position: absolute;
		height: 50%;
		content: attr(content);
		overflow: hidden;
		color: white;
		z-index: 999;
	}

```

但是 H5显示正常 部署到小程序  就是不行 一直读不到 ::before 的content

于是查看小程序API 确认不支持 attr 属性
